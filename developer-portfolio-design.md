Design a modern **scroll-driven developer portfolio UI** with immersive animations and futuristic styling. The portfolio should use a **dark theme, smooth transitions, and storytelling through scrolling**.

GENERAL STYLE
* Animations: smooth scroll-triggered transitions, subtle glow effects, parallax motion
* The entire portfolio should feel **interactive, premium, and dynamic rather than static sections**.

HERO SECTION
Create a full screen hero section that immediately feels interactive and visually impressive.

Layout

* Large bold heading with developer name in the center
* Subtitle describing specialization (AI engineer / backend developer / problem solver)
* Two CTA buttons: “View Projects” and “Scroll to Explore”

Background

* Dynamic digital grid or neural network pattern
* Floating tech icons and particles

Scroll Interaction

* When scrolling starts, the hero title splits into layers and moves slightly apart
* Background grid tilts with a parallax effect creating depth
* Subtitle fades out while the next section smoothly emerges
* Floating particles move slowly as the user scrolls

The hero should feel **alive and immersive rather than a simple landing section**.

EXPERIENCE SECTION
Design a **spiral scroll timeline** for professional experience.

Layout

* A glowing spiral line starting from the top center and expanding downward
* The spiral acts as a path for experience nodes
* Circular nodes placed along the spiral representing each experience

Content per node

* Company name
* Role title
* Duration
* Short achievement summary

Scroll Interaction

* As the user scrolls, the spiral line gradually draws itself
* When a node reaches the center of the screen it activates
* The experience card expands beside the node with a smooth slide + fade animation
* Previously viewed nodes shrink slightly but remain visible

Visual details

* Thin glowing spiral path
* Active node glows with accent color
* Subtle particle background

The section should feel like **scrolling through a story of career growth**.

SKILLS SECTION
Create a visually engaging skills section that avoids traditional boxes.

Layout

* Skills represented as floating circular nodes forming a **constellation or galaxy network**
* Nodes connected with thin glowing lines
* Skills grouped into categories such as Programming, Frameworks, Tools, AI/Data

Scroll Interaction

* As the user scrolls, the skill network slowly rotates like a galaxy
* Different skill groups move toward the center one by one
* When a group reaches the center, its nodes expand slightly and labels appear
* The connections between nodes glow subtly during activation

Animation

* Smooth parallax rotation
* Node scaling on focus
* Subtle glowing particles in background

The section should feel **dynamic and visually engaging while scrolling**.

ACADEMIC BACKGROUND SECTION
Design a scroll-driven academic journey visualization.

Layout

* A diagonal staircase style timeline across the screen
* Each step represents one academic milestone

Content per step

* Institution name
* Degree or qualification
* Duration
* Short description

Scroll Interaction

* As the user scrolls, each step rises upward like climbing stairs
* The active step expands and highlights
* Previous steps fade slightly but remain visible
* Connecting lines animate between steps to show progress

Visual Style

* Clean structured layout
* Subtle technical grid background
* Highlight colors using neon blue accents

The section should visually represent **growth and learning progression**.

CERTIFICATIONS SECTION
Create a scroll-driven certification showcase.

Layout

* Certifications displayed **two cards per row centered on screen**

Card content

* Certification logo
* Certification name
* Issuing organization

Scroll Interaction

* When scrolling down:

  * The current pair of cards split apart
  * Left card slides smoothly to the left
  * Right card slides smoothly to the right
* As they move away, the next pair of certifications pops up from the center with a slight scale and fade animation
* The transition repeats for each pair of certifications

Design

* Modern tech cards with subtle gradient
* Soft glow on hover
* Smooth card transitions

The section should feel **fluid and interactive instead of a static list**.

OVERALL EXPERIENCE

* Smooth scroll storytelling between sections
* Subtle background animations across the page
* Consistent neon tech aesthetic
* Minimal clutter with strong visual hierarchy

The final design should look like a **premium modern developer portfolio with immersive scroll-driven storytelling**.
